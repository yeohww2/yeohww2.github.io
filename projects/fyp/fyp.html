Overview

Project Background

The Sunway Seasonal Parking System is designed to improve the seasonal parking experience at Sunway University. The system allows users to register, manage vehicle information, book parking slots, and access parking-related features in one mobile application. It also supports functions such as QR code validation, penalty balance top-up, feedback, and admin management.

Project Summary

  • Main Idea A university-focused mobile parking system for students and staff to manage seasonal parking more efficiently.
  • Tools Used Flutter, Firebase, Figma, Canva, Stripe, EmailJS, and Word/PDF documentation.
  • Purpose To provide a more organized, convenient, and user-friendly seasonal parking management system.
